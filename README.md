## Sobre o Projeto

Este repositório contém o código-fonte do meu portfólio pessoal. Desenvolvido para apresentar minha trajetória como estudante de **Ciência da Computação** e **desenvolvedora em formação**, o site reúne meus principais projetos acadêmicos, habilidades técnicas e experiências em desenvolvimento web e UI/UX Design.

A interface foi projetada com foco em uma experiência limpa, moderna e acessível.

## 🚀 Funcionalidades em Destaque

* **Dark Mode & Light Mode:** Suporte à alternância de temas visuais para maior conforto do usuário.
* **Suporte Multi-idioma:** Interface preparada para internacionalização (Botão de alternância `EN` / `PT`).
* **Foco em Acessibilidade Web:** Estruturação pensada para que o conteúdo seja simples e funcional para todos os públicos.

## 📂 Organização do Site

O portfólio está estruturado em seções modulares:

1.  **Início:** Tela de boas-vindas com apresentação profissional e acesso rápido aos projetos.

2.  **01. Sobre mim:** Resumo da minha jornada acadêmica, motivações no universo tech e interesses extracurriculares.

3.  **02. Habilidades:** Divisão clara do meu ecossistema de aprendizado em três pilares (Front-end, UI/UX Design e Programação).

4.  **03. Projetos:** Portfólio de aplicações práticas, incluindo:
    * *Redesign Jornal do Commercio* (Foco em Experiência do Usuário e UI).
    * *Sistema de Treinos Hyrox* (Desenvolvido em Python com manipulação de dados).
  
5.  **04. Contato:** Área dedicada para conexões e propostas de oportunidades.
